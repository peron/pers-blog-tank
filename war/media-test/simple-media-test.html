CSS @media test

This is a web page to test the @media CSS rule.

Introduction (Slide 1)

The @media rule can be used in CSS to change the layout of web pages when displayed on different media, such as printers, TVs, braille and audio devices. This web page uses the @media rule to change the layout for presentation, or projection, mode.

Company logo (Slide 2)

We want to display the main header, and the logo at the bottom, on ever slide.
With some CSS tricks, we've displayed the main header and the logo on every slide.

Switching content (Slide 3)

You can have diffenrent content on screen, and in the presentation.
By having two class, say screen and projection, you can easily turn on or off content in the different modes with display: block or display: none, respectively.

Switch to projection mode (Only on screen)

Switch to full screen, or presentation/projection, mode in your browser. In Opera this is done be pressing F11.
